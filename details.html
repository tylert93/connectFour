Overview

This is a recreation of the two-player game Connect Four. To win Connect Four you must be the first player to get four of your coloured checkers in a row either horizontally, vertically or diagonally. In the real-life version of the game, the grid is positioned vertically and therefore players can only take spaces which are either on the bottom row or are above a space which has already been taken.
When a player clicks on a space, the ‘.classList.conatins()’ method is used to check whether this space and the space below have a class of ‘taken’. If the space is not available, the player is alerted that they can go there. If it is available, the ‘.classList.add()’ method is used to give the space the classes of ‘taken’ and either ‘player-one’ or ‘player-two’ depending whose turn it is.
All of the possible four-in-a-row winning combinations are stored in an array. When ever a new space is taken, a function is called which checks if either player has four-in-a-row. A ‘for loop’ and the ‘.classList.contains()’ method are used to run through each winning combination and check if all the spaces in any one combination have the class of ‘player-one’ or ‘player-two’. If a player is found to have won, the player is alerted, and the game is stopped.
